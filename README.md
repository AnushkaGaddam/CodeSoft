#  CODSOFT AI Internship Projects

This repository contains the projects completed as part of the CodSoft Artificial Intelligence Internship.

## 🚀 Projects Included

### 1️ Chatbot
- A rule-based chatbot built using Python
- Responds to user queries using conditional logic

### 2️ Tic Tac Toe (GUI)
- Interactive Tic Tac Toe game using Tkinter
- User vs Computer gameplay with simple AI logic

### 3 Face Detection
- Detects faces in video using OpenCV
- Displays bounding boxes and counts number of faces
- Shows real-time detection messages

## Technologies Used
- Python
- OpenCV
- Tkinter

##  Objective
To understand the basics of Artificial Intelligence concepts such as:
- Rule-based systems
- Game logic and decision making
- Computer vision
